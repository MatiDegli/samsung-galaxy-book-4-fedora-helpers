# Notes

This repository is a small convenience layer around existing Linux fix and workaround projects for Samsung Galaxy Book4 hardware.

The speaker toggle scripts are optional manual helpers for enabling or disabling the speaker fix without digging through the original project files each time.

The fingerprint setup uses the non-official `tenseventy7/libfprint-acer-egismoc` COPR. Treat it as a workaround and review it with the same caution you would use for any third-party package source.
